# F-TREE Tree

Demonstrates the Phase 7 tree feature contract:

- dedicated tree column controls
- expand/collapse public API
- lazy children via `DataSource.getChildren`
- descendant checkbox cascade with mixed parent state
- ancestor-preserving client filter policy
- sibling-preserving client sort policy
- server tree requests that receive filter and sort models

Use the vanilla page at `/#F-TREE` for direct browser verification.
